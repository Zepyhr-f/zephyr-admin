# 00-项目简介

本目录用于存放“全项目统一口径”的基础信息：版本矩阵、文档规范、技术栈总览。  
如果你不确定从哪里开始读，建议先看本目录。

---

| 文件 | 关键词（便于 AI 检索） | 说明 |
|---|---|---|

## 阅读建议
- 新同学先看《版本与环境矩阵》建立全局版本口径。
- 准备写文档或重构结构时，先看《文档规范与模板》。
- 想快速了解整体技术选型时，再看《技术栈总览》。

---

## 维护规则
- 当版本号、端口、依赖环境有变更：必须更新《版本与环境矩阵》
- 新增重要文档规范：必须更新《文档规范与模板》
- 本目录文档被其他文档广泛引用时，应优先作为 SSOT 维护，避免在外围文档复制版本和规则
