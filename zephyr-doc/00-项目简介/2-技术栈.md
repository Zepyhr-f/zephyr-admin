## 核心技术栈清单

### 1. 前端技术栈 (Frontend Stack)
- **核心框架**: React 19
- **脚手架与构建**: Vite
- **UI 框架**: Ant Design Pro 6
- **状态管理**: Zustand
- **异步数据流**: TanStack Query (React Query)
- **网络请求**: Axios + OpenAPI 代码生成
- **编程语言**: TypeScript 严格模式
- **代码规范**: ESLint + Prettier + Husky + lint-staged

### 2. 后端技术栈 (Backend Stack)
- **核心语言**: Java 21
- **基础框架**: Spring Boot 3.4.x
- **微服务治理**: Spring Cloud 2023.0.x
- **配置与注册中心**: Nacos
- **持久层框架**: MyBatis-Plus
- **数据库**: MySQL 8.0 或 OceanBase
- **缓存中间件**: Redis 7.2
- **规则引擎**: AviatorEvaluator
- **智能化引擎**: Spring AI
- **向量存储**: Milvus
- **认证鉴权**: Spring Security + JWT
- **对象存储**: MinIO / 阿里云 OSS

### 3. 基础设施与 DevOps (Infrastructure)
- **本地开发环境**: OrbStack (macOS 容器方案)
- **网关与代理**: Nginx
- **容器化**: Docker + Docker Compose
- **包管理工具**: Maven (后端) / pnpm (前端)
- **持续集成/部署**: GitHub Actions / GitLab CI/CD
