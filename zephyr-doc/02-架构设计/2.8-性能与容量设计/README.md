## 维护规则
- 本目录维护容量口径、压测方法和优化清单，不替代具体监控看板或线上事故复盘。
- 当核心 SLA、压测对象或验收阈值变化时，必须同步更新本目录文档。
- 性能优化建议应与可观测性、数据库和网关设计中的约束保持一致。
