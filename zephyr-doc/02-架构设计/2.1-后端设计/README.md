## 阅读建议
- 想了解整体技术选型时，先看《后端技术栈总览》。
- 想理解模块边界和代码结构时，再看《后端工程结构设计》。
- 涉及请求入口、安全前置与透传约定时，重点看《网关设计》。

## 维护规则
- 本目录主要维护后端长期稳定的技术边界，不记录临时开发任务。
- 版本口径与环境要求如有变化，应同步引用并对齐《版本与环境矩阵》。
- 认证鉴权主题与安全口径冲突时，以安全目录中的 SSOT 文档为准。
