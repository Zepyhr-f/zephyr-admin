# 项目启动与联调说明

本文档旨在说明如何快速在本地环境启动 Zephyr 全栈项目并进行前后端联调。

## 1. 快捷联调方式 (推荐)

在项目根目录下，我们提供了统一的启停脚本 `z-run.sh`。

### 1.1 一键启动
```bash
./z-run.sh start
```
该命令会自动：
1. 在后台启动所有后端微服务容器 (Gateway, System 等)。
2. 在后台拉起前端 Vite 开发服务器。

### 1.2 一键停止
```bash
./z-run.sh stop
```

### 1.3 一键重启
```bash
./z-run.sh restart
```

## 2. 独立模块调试

### 2.1 后端调试 (Java)
- 环境要求：JDK 17+, Maven 3.8+, Docker (或各组件独立安装)。
- 调试建议：使用 IntelliJ IDEA 分别启动 `zephyr-system` 和 `zephyr-gateway`。

### 2.2 前端调试 (React)
- 进入目录：`cd zephyr-web`
- 安装依赖：`pnpm install`
- 启动服务：`pnpm dev`
- 浏览器访问：`http://localhost:8000`

## 3. 日志查看
- 后端日志：`logs/backend.log`
- 前端日志：`logs/frontend.log`
