## 维护规则
- 新增功能设计前，先更新本目录或子目录 README，登记文件、关键词和状态。
- 功能总览文档用于管理模块边界和导航，细节变更应同步落到对应详细设计文档。
- 与安全、鉴权、数据库执行顺序等横切主题冲突时，以对应 SSOT 文档为准。
