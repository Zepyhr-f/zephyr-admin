## 阅读建议
- 先看《系统功能设计总览》建立模块地图。
- 进入具体能力前，再按系统管理、安全审计、监控、基础设施、开发工具逐项阅读。

## 维护规则
- 本目录维护平台通用功能，不承载业务域专属方案。
- 横切安全、数据库和接口规范冲突时，应回链到对应架构设计或数据库文档。
- 新增系统功能时，必须先更新《系统功能设计总览》和本 README。
