# v1.0.8 更新日志

**发版时间**: 2026-04-16

## 后端变更 (zephyr-server)
- **代码生成器加固**：修复了 Spring Boot 3 下控制器参数反射异常导致的 500 错误。
- **ZIP 下载实现**：完成了代码生成后的压缩包流式下载功能。
- **中文乱码修复**：全局 JDBC 连接串增加 `characterEncoding=utf8`，彻底解决数据库元数据中文乱码。
- **接口文档聚合**：网关集成 Knife4j Gateway，实现微服务 API 文档统一入口。
- **服务治理优化**：规范化 Nacos 服务注册名为 `zephyr-system` 和 `zephyr-generator`。

## 前端变更 (zephyr-web)
- **列表 UI 优化**：使用 `TableDropdown` 收纳低频操作按钮，优化列宽，提升表格整洁度。
- **代码预览增强**：预览窗口增加“一键复制”功能，并优化了 Tab 标签显示。
- **路由清理与恢复**：按需调整了“系统接口”菜单，确保 Knife4j 交互式文档正确嵌入。

## 文档变更 (zephyr-doc)
- **发版规范执行**：新增 v1.0.8 版本更新日志。
