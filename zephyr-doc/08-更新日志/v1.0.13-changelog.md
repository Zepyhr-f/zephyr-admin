# v1.0.13 更新日志

**发版时间**: 2026-04-18

## 后端变更 (zephyr-server)
- 本次发布无后端代码变更。

## 前端变更 (zephyr-web)
- **UI 结构重构**：实现了“用户管理”页面的“分体式（Split-Box）”布局，将查询条件与查询结果拆分为独立的卡片容器。
- **查询组件优化**：
    - 实现了动态响应式 Span 列宽（收起时 3 列，展开时 4 列）。
    - 统一减小了查询表单的标签宽度与输入框尺寸，使界面更加精炼和谐。
    - 自定义了展开后的操作按钮布局，使其独立成行并保持右对齐。
- **操作栏定制**：为“用户管理”设计了全新的按钮聚合组，集成了新增、导入、导出、批量删除功能，并配有深浅色自适应的方形背景框。
- **监控页面统一样式**：完成了“定时任务”、“服务监控”与“缓存监控”页面的空间压缩，移除了冗余标题区域。
- **系统稳定性修复**：通过改用图标直接路径引用与清理 `.umi` 编译缓存，解决了部分环境下 “Module factory not available” 的加载故障。

## 文档变更 (zephyr-doc)
- 新增 `v1.0.13-changelog.md` 版本日志。
- 更新了前端布局参考设计方案。
