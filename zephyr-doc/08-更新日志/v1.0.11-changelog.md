# v1.0.11 更新日志

**发版时间**: 2026-04-18

## 后端变更 (zephyr-server)
- **网关安全**: 将 JWT 令牌过期时间从 1 小时延长至 24 小时，解决了因 Token 过期导致的频繁 401/500 系统报错问题。
- **配置优化**: 更新了 `zephyr-gateway` 的 `application.yaml` 配置并完成了网关服务的重启应用。

## 前端变更 (zephyr-web)
- **路由清理**: 移除了冗余的 “欢迎页” 和 “异常页” (403, 404, 500)，使导航结构更加聚焦核心业务。
- **顶栏重构**:
  - 重新排列了 Header 操作按钮，将 **白天/黑夜切换模式** 按钮移至 **语言切换按钮** 旁边，优化了交互路径。
  - 补全了 Header 的搜索、消息通知、系统设置等功能图标，提升了视觉一致性和功能完整性。
  - 使用 `SunOutlined` (金黄色) 和 `MoonOutlined` 图标增强了主题切换的视觉提示。
- **布局微调**: 修正了部分自定义布局中的间距和层级问题，确保双列菜单与自定义 Header 的衔接自然。

## 文档变更 (zephyr-doc)
- **版本记录**: 新增 `v1.0.11` 更新日志文件，记录本次架构修正与安全优化的详细变动。
