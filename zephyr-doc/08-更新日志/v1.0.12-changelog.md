# v1.0.12 更新日志

**发版时间**: 2026-04-18

## 后端变更 (zephyr-server)
- 本次发版未涉及后端核心逻辑变动。

## 前端变更 (zephyr-web)
- **主题系统升级**: 引入了基于 CSS 变量的全局主题系统，实现了真正意义上的“极简纯白 (Light)”与“极客纯黑 (Dark)”模式切换。
- **配色一致性**: 修复了黑夜模式下标签栏 (MultiTabs)、卡片、表格及输入框底色不一致的问题，确保背景色强制为 `#000000`。
- **国际化简化**: 
  - 移除了多余的语言包，仅保留中、英两种核心语言。
  - 将下拉式语言切换重构为 **一键直切** 按钮，并增加了动态文字提示 (`EN` / `中`)，大幅提升操作效率。
- **交互细节优化**: 
  - 适配了全黑背景下的面包屑分隔符及各功能图标的可见性。
  - 优化了标签栏在不同主题下的边框过渡效果。

## 文档变更 (zephyr-doc)
- **版本记录**: 新增 `v1.0.12` 更新日志，详述了主题系统与国际化交互的重构细节。
