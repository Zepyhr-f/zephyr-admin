# Zephyr 系统更新日志 (v1.0.3)
日期: 2026-04-16

## 核心更新
- **系统管理模块 (RBAC) 全面实现**：
  - **用户管理**：支持用户增删改查、状态切换、重置密码及角色分配。集成了与部门的关联查询。
  - **角色管理**：支持角色增删改查、状态切换及权限映射（菜单分配）。
  - **菜单管理**：实现了递归树形结构，支持前端菜单导航的层级渲染。
  - **部门管理**：新增部门树形维护功能，支持层级部门管理。

- **后端架构与安全性修复**：
  - **网关权限绕过修复**：解决了 `AuthFilter` 中路径匹配逻辑错误导致的 401/403 异常。现在网关仅负责身份认证，细粒度权限由业务端控制。
  - **数据类型一致性修复**：统一了 VO 中 `createTime` 的类型为 `java.util.Date`，解决了属性拷贝时的类型不匹配异常。
  - **编译兼容性优化**：为 Controller 参数显式增加了 `@RequestParam(name = "...")`，解决了在无 `-parameters` 编译参数下的参数绑定失败问题。

- **前端功能集成**：
  - 基于 Ant Design Pro 完成了系统管理全量页面的开发。
  - 封装了完整的系统管理 API 服务层。
  - 配置了模块化的路由与国际化资源。

## 技术侧重点
- **后端**：Spring Cloud Gateway 过滤器优化、MyBatis-Plus 树形查询、ZBeanUtils 属性拷贝兼容性。
- **前端**：ProTable 高级表格封装、Transfer 角色分配组件、TreeSelect 部门选择器。

## 待办事项
