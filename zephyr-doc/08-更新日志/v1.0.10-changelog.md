# v1.0.10 更新日志

**发版时间**: 2026-04-18

## 后端变更 (zephyr-server)
- 无核心逻辑变更。

## 前端变更 (zephyr-web)
- **布局重构**: 将全局布局模式从 `mix` 切换为 `side` (侧边菜单) 模式，提升后台专业感。
- **自定义 Header**: 在右侧页面内容区顶部实现了自定义顶栏，集成了 `HeaderBreadcrumb` 文字导航及用户信息展示。
- **多标签页集成**: 开发并注入了标签页（Multi-Tabs）系统，锁定在顶栏下方，支持多页面快速切换。
- **视觉去重**: 移除了顶栏左侧的多余 Logo，解决“双 Logo”显示问题；禁用了侧边栏底部重复的头像条。
- **面包屑逻辑增强**: 优化了 `HeaderBreadcrumb` 组件，增加了对自定义 Header 的数据适配及 URL 解析兜底。

## 文档变更 (zephyr-doc)
- **架构文档**: 新增 `03-架构设计/3.2-前端设计.md`，深度解析了新版 Layout 与 Multi-Tabs 的技术路线。
- **设计模型**: 更新了数据库设计定义文件 `1.yaml`。
- **发版记录**: 新增本次 v1.0.10 的版本更新记录。
