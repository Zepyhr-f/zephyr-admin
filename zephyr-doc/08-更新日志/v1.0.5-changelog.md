# v1.0.5 更新日志

**发版时间**: 2026-04-16

## 后端变更 (zephyr-server)
- **安全审计核心实现**:
  - 新增 `SysLoginLog`、`SysOperLog` 实体类、Mapper、Service 及 Controller。
  - 新增 `OnlineUserController`，支持基于 Redis 的实时在线用户监控与强制退功能。
- **公共组件增强**:
  - `RedisUtil`: 新增 `keys` 模式匹配方法，修复 `getSet` 逻辑缺陷。
  - `pom.xml`: 补全 `zephyr-system-biz` 对 `zephyr-starter-jwt` 的依赖。
- **环境治理**: 修复了数据库中缺失审计表的问题，同步执行了初始化 SQL。

## 前端变更 (zephyr-web)
- **安全审计模块上线**:
  - 新增“登录日志”页面，支持分页、搜索与批量逻辑删除。
  - 新增“操作日志”页面，提供详尽的请求/回包 JSON 详情查看。
  - 新增“在线用户”监控页面，集成会话实时查看与一键强退功能。
- **导航与体验**:
  - 在 `routes.ts` 中集成安全审计一级菜单。
  - 补全了安全审计相关的中英文国际化 (i18n) 文档。

## 文档变更 (zephyr-doc)
- **项目定义修正**: 修正了 `project.md` 中对前端技术栈的描述（明确为 Ant Design Pro / React / Umi）。
- **更新日志**: 生成 v1.0.5 发版记录。
